# AasraGuide-AI StudentSupportSystem

AasraGuide is a smart AI-powered student productivity system built using Flask, designed to streamline daily learning tasks, track academic progress, and provide AI-generated study assistance.
The project provides a private, secure, personalized dashboard for every student with:

Smart Chat Assistant
Assignment Manager
Notes Manager with Rich Editor
Pomodoro Focus Timer
AI Exam Helper
File attachments, reminders, and more!
This is a 1-year, 2-credit academic project, developed with real-world usability in mind.
